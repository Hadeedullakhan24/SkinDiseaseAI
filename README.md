# 🧠 Skin Disease Classification AI

A deep learning-based web application for classifying skin diseases using a hybrid CNN + EfficientNet model.

---

## 🚀 Features

- 🔍 Skin disease classification (5 classes)
- 🧠 Hybrid Model (CNN + EfficientNet)
- 📊 Confidence scores & probabilities
- 🔥 Grad-CAM visualization (model explainability)
- 🌐 Interactive UI using Streamlit

---

## 📂 Dataset

- HAM10000 Dataset (Skin lesion images)

---

## 🏗️ Project Workflow

1. Data Preprocessing
2. CNN Model Training
3. Transfer Learning (EfficientNet)
4. Model Evaluation
5. Hybrid Model (CNN + EfficientNet)
6. Deployment using Streamlit

---

## 🧪 Classes Used

- Actinic Keratosis
- Basal Cell Carcinoma
- Benign Keratosis
- Melanoma
- Nevus
